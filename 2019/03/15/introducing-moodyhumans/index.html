Moody Humans v0.1.0


Moody Humans was a simple “mood logging using scientific measurement tools” project.

I primarily wanted to have some experience with web development using Vue. The backend is built using Django.

The fanciest feature of Moody Humans is that it’s integrated to Open Humans. Which opens up the benefits of OpenHumans ecosystem, chief among them being the personal data notebooks.

Head over to Moody Humans to take a look!
